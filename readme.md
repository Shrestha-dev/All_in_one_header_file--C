# C PROJECT

## 🚀 **DESCRIPTION**
*This project contains C program that is used to build a all in one header file.*
*This header file contains the mixture of all the pre build libraries of various C header files, only adding this file as a main header file will complete all the requirements and your need not to add separate header files for various operations*

---

### 📌**The Header Files and their corresponding Libraries added are as follows:**
- Ctype.h
- String.h
- Math.h
- Stdlib.h
- Stdio.h

---

## 💻**How to run in VS Code Terminal:**
```bash
gcc filename.c -o filename

./filename

   or

gcc filename.c

./a.exe
```
